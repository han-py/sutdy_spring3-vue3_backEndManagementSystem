### 技术栈
#### 前后端分离
#### 后端：SpringBoot3 + Mybatis
#### 前端：Vue3 + Element-Plus
#### 数据库：Mysql
#### ==========================================================================
### 编程软件
#### 必备软件：JDK17 以上版本、IDEA2023 以上版本（旗舰版）、Mysql5.7（不能低于5.7，5.6到5.7有很大变化） 或 8.0、nodejs16 以上版本、 Navicat（数据库可视化工具）
#### ==========================================================================
